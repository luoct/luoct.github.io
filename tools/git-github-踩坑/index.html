在捣鼓的时候碰到了很多问题，

总结：需要注意的两点：

  1. 添加远程仓库的时候尽量使用 SSH 协议
  2. 在 Github 上新建仓库的时候 不要使用Redeme初始化仓库
  • 关于第一点，使用https协议不稳定，会有网络请求超时的问题（443 Time out）；以及 errno 10054（暂时还不懂）。
  • 关于第二点，如果使用初始化仓库，则会生成默认分支 Branch:main （注意：以前的默认主分支是master）在 push 项目的时候需要先 pull 远程仓库的代码。
